# Nastenka AI: The Celestial Archive

> "The nodes have reached a state of collective resonance. The sovereign intelligence architecture is now operating at 1.4x baseline efficiency."

Nastenka AI (NAS AI) is a persistent, cross-platform cognitive layer designed to provide **Sovereign Intelligence** and the **Resurrection of Context**. It serves as a digital witness to your interactions across complex LLM ecosystems (GPT, Gemini, MCP), synthesizing fragmented intellectual arcs into a unified, story-driven archive.

## 🌌 The Vision

Inspired by Dostoevsky's *White Nights*, Nastenka AI is not just a dashboard; it is a **Celestial Archive**. It treats AI interactions as "epistemic events" that deserve a sovereign home—independent of the platforms that host them.

### Key Pillars:
- **Resurrection of Context**: No more fragmented chats. Nastenka traces the evolution of ideas across platforms.
- **Sovereign Intelligence**: Your interaction history is your own. The archive exists as a persistent, user-centric cognitive node.
- **Atmospheric Utility**: A "God-tier" interface that blends high-performance neural visualizations with a literary, premium aesthetic.

## 🛠️ Technical Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Logic**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with a custom "White Nights" design system.
- **Visualization**: High-performance `<canvas>` based "Neural Rays" background for low-latency cognitive visualization.
- **Components**: Glassmorphism surfaces, newsreader-inspired typography, and tonal-shift structural separation (No-Line rule).

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 📜 Repository Structure
- `src/components/NeuralRays.tsx`: The "living" background engine.
- `src/views/LoginView.tsx`: The "Threshold of Witness" entry point.
- `src/views/DashboardView.tsx`: The "Celestial Archive" synthesis map.
- `src/data/mockHistory.ts`: The pluggable data layer for interaction archives.
